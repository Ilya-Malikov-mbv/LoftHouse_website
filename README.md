# LoftHouse — Элитный жилой комплекс 🏢

Полностью адаптивный лендинг для жилого комплекса премиум-класса в Санкт-Петербурге. Проект выполнен с использованием компонентной архитектуры стилей (SASS/SCSS) по методологии БЭМ и интерактивными интеграциями на чистом JavaScript.

---

## 🚀 Функционал и особенности

- **Адаптивная навигация:** Кастомное мобильное меню ("бургер") с блокировкой прокрутки экрана (`no-scroll`) при открытии.
- **Интерактивная карта:** Интеграция **Yandex Maps API** со стилизованным кастомным маркером (location pin) и информационным балуном (адрес и телефон ЖК).
- **Удобные формы:** Маскирование ввода номера телефона для удобства пользователей и валидация плейсхолдеров на чистом JS.
- **Премиальный дизайн:** Использование современных SASS-миксинов для гибкой адаптивности и чистой типографики.

---

## 🛠️ Стек технологий

- **Markup:** HTML5 (Семантическая кроссбраузерная верстка)
- **Styles:** SASS/SCSS (Структурированная архитектура: разделение на базовые стили `base/` и независимые компоненты `blocks/` по БЭМ)
- **Scripts:** Vanilla JavaScript (ES6+), Yandex Maps API, Phone Mask Library
- **Сборка:** Компиляция SCSS в CSS

---

## 📁 Структура папок (Стили)

Код стилей разбит на логические модули для удобной поддержки:

```text
└── scss/
    ├── base/
    │   ├── _base.scss       # Базовые стили страницы
    │   ├── _mixins.scss     # Переиспользуемые миксины (адаптивность, шрифты)
    │   ├── _reset.scss      # Сброс стандартных стилей браузера
    │   └── _vars.scss       # Цветовая палитра и шрифтовые переменные
    ├── blocks/              # Компоненты сайта (header, apartments, benefits, video, map и др.)
    └── main.scss            # Главный файл-диспетчер (импорт всех модулей)
```
